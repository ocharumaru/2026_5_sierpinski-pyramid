複素数平面 $\mathbb{C}$ 上において、スケール比 $r = \frac{1}{3}$ とし、回転と平行移動を伴う4つの縮小写像からなるIFSで定義します。

$f_1(z) = \frac{z}{3}$

$f_2(z) = \frac{z}{3} e^{\frac{i\pi}{3}} + \frac{1}{3}$

$f_3(z) = \frac{z}{3} e^{\frac{-i\pi}{3}} + \frac{1}{2} + \frac{\sqrt{3}}{6} i$

$f_4(z) = \frac{z}{3} + \frac{2}{3}$

初期集合を区間 $[0, 1]$ とし、これら4つの写像を反復適用した極限がコッホ曲線となります。
